# Support

## Where to ask for help
- Usage questions: open a GitHub Discussion (recommended).
- Bug reports: use the bug report issue template.
- Feature ideas: use the feature request issue template.

## What to include
When asking for help, include:
- Your OS and Python version.
- How you installed `quickthink`.
- The exact command you ran.
- Full error output.
- Whether Ollama is running and which model you used.

## Quick self-check
- Verify install: `quickthink list-models`
- Verify tests: `PYTHONPATH=src .venv/bin/pytest -q`
- Verify Ollama: `ollama list`

## Security issues
For suspected vulnerabilities, do not file a public issue. See `SECURITY.md`.
